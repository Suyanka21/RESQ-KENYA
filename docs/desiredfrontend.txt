ResQ Emergency App - Codebase Description
Overview
ResQ is a fully-functional emergency assistance web application designed for Kenyan motorists. It operates as an "Uber for Rescue" - connecting stranded drivers with verified rescue units for roadside emergencies. The app is built with React/TypeScript and styled with Tailwind CSS using a custom dark theme.

Application Structure
Entry Point: app/page.tsx imports App.tsx

Main State Management: The app uses a simple ViewState ('landing' | 'dashboard') in App.tsx to toggle between the marketing landing page and the interactive dashboard.
